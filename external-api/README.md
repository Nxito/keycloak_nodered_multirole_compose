# External-api

Servicio de gestión de apis a terceros
Guarda en caché los ultimos datos recibidos de distintos servicios

## Los servicios de node-red requieren los siguientes archivos

-  package.json
-  flows.json
-  flows_cred.json
-  settings.js
-  static folder
